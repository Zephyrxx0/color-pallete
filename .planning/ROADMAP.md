# Project Roadmap

**Project:** Color Palette Plugin for Obsidian
**Version:** 1.0.0

## Milestone 1: MVP - Core Features

### Phase 1: Core MVP
- [x] Card creation, display, and deletion
- [x] 2–4 color blocks per card
- [x] Click-to-copy (hex only)
- [x] Basic settings (copy format)
- [x] Persist to plugin data
- [x] Card renaming capability
- [x] Multi-color copy with shift+click
- [x] Confirm delete modal

### Phase 2: Polish
- [ ] Dynamic CSS / theme integration
- [ ] Card naming (auto + manual)
- [ ] Optional block labels with label color
- [ ] Toast notifications
- [ ] Enhanced UI/UX

### Phase 3: Organization
- [ ] Groups / categories
- [ ] Drag-and-drop card reorder
- [ ] Search / filter in toolbar
- [ ] Collapse groups

### Phase 4: Power Features
- [ ] Full settings panel (all options)
- [ ] Import / Export JSON
- [ ] All copy formats (RGB, HSL, CSS var, Tailwind)
- [ ] Keyboard navigation & accessibility audit
- [ ] Mobile testing pass
